1. git status - Shows the status of the working directory and staging area.
2. git add <file> - Adds a file to the staging area.
3. git commit -m "<message>" - Commits the staged changes with a message.
4. git push - Pushes changes to a remote repository.
5. git pull - Fetches and integrates changes from the remote repository.
6. git branch - Lists, creates, or deletes branches.
7. git checkout <branch> - Switches to a different branch.
8. git merge <branch> - Merges changes from one branch into another.
9. git log - Shows the commit history.
10. git clone <repo-url> - Clones a repository from GitHub.
